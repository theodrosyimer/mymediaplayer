## Usage

**Install**

```sh

```

**Dev**

```sh
npm run dev
```

**Build**

```sh
npm run build
```

**Preview**

if the project is already built:

  ```sh
  npm run preview
```

otherwise do both:

```sh
  npm run build && npm run preview
```
